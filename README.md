# Team Topology Classifier
Par [Collaboration Solved](https://collaborationsolved.com) — Pierre-Cyril Denant

Outil self-service pour classifier vos équipes selon Team Topologies, mapper votre écosystème, et identifier les gaps Current → Target.

## Déploiement
```bash
npm install
npm run dev      # local
npm run build    # Vercel
```
Vercel : push GitHub → Import → Framework = Vite → Deploy.
